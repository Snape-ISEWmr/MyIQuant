# 项目概述

![项目全景](./images/02-project-overview.png)

## 项目目标
国信iQuant量化策略交易平台是一个专业的量化交易策略开发与执行平台，提供多种策略模板和回测框架，支持实盘交易和策略研究。

## 核心用户
- **量化交易员**：开发和执行自动化交易策略
- **策略研究员**：进行策略回测和优化
- **投资机构**：多策略组合管理和风险控制

## 系统边界
- **iQuant平台**：国信证券量化交易平台核心引擎
- **通达信接口**：通过iQuantTDX提供行情和交易接口
- **vnpy框架**：部分策略使用vnpy作为回测引擎
- **交易所**：股票、期货、期权市场交易接口

## 核心业务流程
1. **策略开发**：基于模板创建新策略 → 参数配置 → 回测验证
2. **实盘交易**：策略初始化 → 行情订阅 → 信号生成 → 订单执行 → 风控检查
3. **策略监控**：持仓监控 → 盈亏统计 → 风险预警
